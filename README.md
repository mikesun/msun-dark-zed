## Bold style
I use bold for some syntax:

Zed `settings.json`
```json
{
 	"experimental.theme_overrides": {
		"syntax": {
			"function": {
				"font_weight": 700
			},
			"operator": {
				"font_weight": 800
			},
			"type": {
				"font_weight": null
			}
		}
	}
}
```
